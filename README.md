# Chegg Scheduling App

A monorepo containing both the frontend and backend for the Chegg Scheduling App.

## Structure

```
chegg-scheduling-app/
├── backend/    # Node.js/Express API with Prisma ORM
└── frontend/   # Frontend application (coming soon)
```

## Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

See [backend/README.md](backend/README.md) for full backend setup instructions.
